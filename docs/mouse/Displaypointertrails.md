# 鼠标指针轨迹设置注册表项

## 功能位置
`控制面板` > `硬件和声音` > `鼠标` > `指针选项` 选项卡

## 注册表路径
HKEY_CURRENT_USER\Control Panel\Mouse

## 键值说明
| 键值名称     | 数据类型 | 取值范围 | 默认值 | 功能描述                 |
|--------------|----------|----------|--------|--------------------------|
| MouseTrails  | REG_SZ   | 0-16     | 0      | 控制鼠标指针轨迹显示效果 |

## 详细说明

### MouseTrails 参数
- ​**0**​：禁用指针轨迹（默认值）
- ​**1-16**​：启用指针轨迹并设置轨迹长度，数值越大轨迹越长

### 修改方法
1. 按 `Win + R` 输入 `regedit` 打开注册表编辑器
2. 导航至上述路径
3. 双击 `MouseTrails` 键值
4. 输入所需数值（0-16之间）
5. 重启计算机使设置生效

### 注意事项
- 该设置仅影响当前用户
- 数值超过16时将会变成正常指针
